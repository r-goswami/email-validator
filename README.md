# email-validator
Go library to verify email addresses through Email verification API provided by 3rd party services as abstractApi, hunter.io etc.

## abstract.com client
### Build
```bash
cd cmd/aapi_validate
go build
```

### Run
```bash
cd cmd/aapi_validate
./aapi_validate validate -apiKey API_KEY -email EMAIL
```
